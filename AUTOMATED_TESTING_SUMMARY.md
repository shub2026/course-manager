## 🔧 使用指南

### 本地运行测试
```bash
# 进入服务端目录
cd /tmp/kec-manager/server

# 安装依赖（如果未安装）
npm install

# 运行所有测试
npm run test:run

# 查看覆盖率
npm run test:coverage

# 打开HTML报告
open coverage/index.html
```

### 运行特定测试
```bash
# 只运行认证测试
npx vitest tests/auth.test.js

# 只运行权限测试
npx vitest tests/rbac.test.js

# 运行包含"登录"的测试
npx vitest -t "登录"
```

### 调试模式
```bash
# 监视模式（文件变化自动重新运行）
npm test

# 详细输出
npx vitest --reporter=verbose

# 单个测试调试
npx vitest tests/auth.test.js --inspect-brk
```
